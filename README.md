# 项目简介

## 核心功能
- 基于YOLOv8的草莓成熟度检测，支持800张数据集，包含['成熟的', '未成熟的', '果梗']三类标注
- 提供完整源码、标注数据集、训练好的模型及训练结果
- 支持多种输入方式：单张图片、图片文件夹、视频文件、本地摄像头
- 界面功能：实时显示目标位置、目标总数、置信度信息；支持批量检测和结果保存
- 可自定义置信度阈值和IOU阈值
- 支持选择显示标签和原图，可切换不同检测模型
- 包含项目运行教程，使用PyCharm+Anaconda或VSCode+Anaconda环境

## 适用场景
- 农业领域草莓成熟度自动化检测
- 智能农业采摘辅助系统
- 农产品质量分级与评估
- 农业图像识别研究

## 技术特点
- 采用PyCharm+Anaconda或VSCode+Anaconda开发环境
- 主要技术栈：Python3.9、OpenCV-Python、PyQt5、Ultralytics、Torch1.9
- 使用YOLOv8深度学习框架进行目标检测
- 提供图形用户界面，操作简便直观
